Cashe - папка с файлами для работы с памятью на диске, в частности с кешем, для более быстрой отработки а так же для защиты
Public - папка со статическими элементами системы(img, css, js и т.п) и точкой входа index
sys - папка с файлами базы движка, содержит программу управления процессами. Он же подключает композер когда нужно. Является ядром сайта
src - в этой папке содержится весь код для подключения и управления всего остального, пользовательский код
Controllers - папка с файлами (распологается в папке sys) управления системой движка, соединяет все папки с файлами, создает обмен информацией. Мост среди всех файлов.
Modules - папка с модулями системы (распологается в папке sys), базовой логики движка сайта
REST - папка с файлами для подключения к внешним источникам путем Rest запросов(находится в папке sys). 
Vendor - папка со сторонними файлами и библиотеками
View - папка с файлами представлений. В ней содержатся файлы с основной разметкой HTML сайтов, со встроенными скриптами.
